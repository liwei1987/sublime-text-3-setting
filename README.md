# sublime-text-3-setting
    sublime text 3 配置同步
    参考 https://packagecontrol.io/docs/syncing

将工程目录下的文件放入以下相应位置

##Mac
    ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/
##Linux
    ~/.config/sublime-text-3/Packages/
##Windows
    "$env:appdata\Sublime Text 3\Packages\"
